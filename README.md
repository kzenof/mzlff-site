# MZLFF — фан-сайт

Неофициальный многостраничный сайт, посвящённый Илье (MZLFF) и его альтер-эго **Илюха реп**.

## Обновление релизов с Яндекс Музыки

Релизы подтягиваются скриптом с API Яндекс Музыки. **Запускать нужно с компьютера в РФ** — серверы GitHub блокируются (HTTP 451). Поэтому используются скрипты в powerShell 
https://kzenof.github.io/mzlff-site - GitHub Pages
