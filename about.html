О сайте

Сайт является неофициальным фан-сайтом, посвящённым исполнителю MZLFF (Илья). Создан в рамках курсовой работы и не монетизируется. Вся информация взята из открытых источников и не является публичной офертой.

Структура проекта

Сайт разделён на несколько страниц: главная, музыка, галерея, биография, ссылки и эта страница. Статические файлы организованы в папки css/, js/, assets/ и data/.

Как обновлять релизы

Карточки альбомов и синглов генерируются из файла data/releases.json. Когда выходит новый релиз, достаточно добавить одну запись:

  • JSON-файл — редактируешь только data/releases.json, HTML не трогаешь. Просто и надёжно для GitHub Pages.
  • Виджет Яндекс Музыки — iframe на странице артиста обновляется сам, но дизайн карточек там не твой.
  • API Яндекс Музыки — можно автоматически тянуть релизы, но нужен сервер или GitHub Action, плюс возможны ограничения API.

Для портфолио и курсовой оптимален первый вариант: карточки выглядят как задумано, а обновление занимает минуту.
